# Card Vault ⚾

**Live:** https://italian1superman.github.io/card-vault/

## Public data on GitHub

| File | What |
|------|------|
| `data/mlb-career.json` | MLB career hitting/pitching (public) |
| App UI | `index.html` + scripts on GitHub Pages |

Your **card collection** stays on your phone (`localStorage`). Use **⋯ → Backup** for a local JSON export — it is not uploaded as a private GitHub collection.

## MLB stats

⋯ → **MLB stats JSON** / **Import MLB stats** loads career lines onto cards from `data/mlb-career.json`, with live MLB Stats API fallback.
