# Card Vault

Sports card collection PWA — values, P/L, CardSight pricing, Excel import/export. All data stays in your browser.

## Open on iPhone

1. Open this site in **Safari** (not Chrome).
2. Tap **Share** → **Add to Home Screen**.
3. Open the **Card Vault** icon.

## Desktop

Open the Pages URL in any browser. Use **⋯ → Backup** often.

## Privacy

No server database. Cards and photos are stored in `localStorage` + IndexedDB on your device.
